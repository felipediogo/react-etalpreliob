# React Etalpreliob
[![All Contributors](https://img.shields.io/badge/all_contributors-4-orange.svg)](#contributors) [![License: MIT](https://img.shields.io/github/license/hyanmandian/react-etalpreliob.svg)](LICENSE)

Just another boilerplate based create-react-app (everything that works with create-react-app works with us) with many useful things like:

- Jest
- Axios
- Rewired
- Rematch
- Storybook
- Hot Loader
- Sanitize.css
- Reach Router
- React Loadable
- Styled Components
